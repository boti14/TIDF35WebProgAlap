Git parancsok

git clone - Olyan parancs, mely segítségével egy repository lemásolható a sajátgépre.

git status - Változások leellenörzésére szolgál, valamint listázza a untracked, modifed és stagelt adatokat.

git add - A módosult fájlokat követésre veszi fel.

git commit - A fájlokon történt változást ezzel a parancsal tudjuk menteni és a mögé írt ("")-ben meg tudjuk nevezni a változást.

git push - Ezzel a paracsal feltöltjük a változásokat a távoli repo-ba.
